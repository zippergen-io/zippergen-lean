# zippergen-lean

Lean 4 formalizations accompanying [zippergen](https://github.com/zippergen-io) papers. The `msc-agents/` directory contains the mechanically verified definitions and proofs for the paper *"Provable Coordination for LLM Agents via Message Sequence Charts"*, including the Uniform Zipper Lemma and the Inductive Correctness Theorem. Build with `cd msc-agents && lake build`.
